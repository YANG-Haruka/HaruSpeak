# HaruSpeak backend dependencies.
#
# Installing (recommended — uv is ~10x faster than pip):
#     pip install uv
#     uv pip install -r backend/requirements.txt
#
# Or plain pip:
#     pip install -r backend/requirements.txt
#
# Notes:
#   - Python 3.11 is required.
#   - On Windows, ffmpeg is needed on PATH for audio decoding. Either
#     `conda install -c conda-forge ffmpeg` or download the static build.
#   - `pyopenjtalk` is OPTIONAL — it enables Japanese pitch-accent
#     annotations but needs a C++ toolchain to build (MSVC on Windows,
#     gcc on Linux). Skip it and the Japanese path falls back to
#     fugashi + unidic-lite (pure wheels, no compiler needed).
